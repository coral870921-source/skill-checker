# 🔍 Skill Checker — Claude Code 技能评估器

> 🛡️ find-skills 的最佳搭档：发现新技能后，一键全面评估 —— 功能、安全、依赖、成本，安装前心里有数。
>
> 装不装？先查一下。

---

## 🤔 为什么需要这个 Skill？

`find-skills` 能帮你找到各种 skill，但装之前你有这些问题：

```
没检查直接装 ❌                           先检查再装 ✅
───────────────                           ───────────────
装完发现要 API Key ──► 浪费时间            提前发现依赖，准备好再装
SKILL.md 500 行 ──► 每次加载吃 token       估算 token 成本，值不值心里有数
skill 里 eval 远程代码 ──► 安全风险         识别安全隐患，提前规避
装完发现和已有 skill 重复 ──► 冗余          对比现有技能，避免重复安装
```

---

## ✨ 检查覆盖 5 个维度

| 维度 | 说明 |
|------|------|
| 🔬 **功能分析** | 核心功能、使用场景、触发频率、与已有技能是否重叠 |
| 🔌 **依赖检查** | 外部 API、MCP Server、CLI 工具、付费服务、环境变量 |
| 🛡️ **安全评估** | 数据泄露、代码执行、文件操作、供应链风险 + 规避方法 |
| 💰 **资源消耗** | Token 估算、内存占用、磁盘空间、网络带宽 |
| ⭐ **综合建议** | 推荐安装 / 谨慎安装 / 不推荐安装 + 理由 |

---

## 📦 安装

```bash
# 一行安装
npx skills add coral870921-source/skill-checker -g -y

# 或手动安装
mkdir -p ~/.claude/skills/skill-checker
cp SKILL.md ~/.claude/skills/skill-checker/
```

**前置条件：**
- [Claude Code](https://docs.anthropic.com/en/docs/claude-code) CLI
- 已安装 `find-skills`（搭配使用，非必须）

## 📋 检查报告样例

```markdown
## Skill 检查报告: browser-act

### 概要
- **功能**：浏览器自动化，模拟用户操作网页
- **实用性**：⭐⭐⭐ 高 — 支持搜索、填表、截图等常见场景
- **安全性**：⭐⭐ 中 — 执行远程 JS，需注意供应链风险
- **资源消耗**：⭐⭐ 中 — 启动浏览器进程，占 200-500MB 内存

### 依赖项
| 依赖          | 类型     | 必须 | 说明                     |
|---------------|----------|------|--------------------------|
| Playwright    | npm 包   | 是   | 浏览器自动化引擎          |
| Chrome/Chromium | 浏览器 | 是   | 内置下载或系统已有        |

### 安全风险
| 风险                | 严重程度 | 规避方法                     |
|---------------------|----------|------------------------------|
| 执行动态 JS 代码     | 中       | 审查 skill 中的脚本内容       |
| 需要浏览器登录态      | 中       | 使用独立 profile，勿存敏感信息 |
| npm 供应链风险       | 低       | 锁定版本，定期 audit          |

### 资源消耗
- Token: 约 800 tokens/次加载
- 内存: 中（浏览器进程 200-500MB）
- 磁盘: ~150 MB（含浏览器）

### 建议
✅ 推荐安装 — 功能实用，风险可控，注意定期更新依赖。
```

## 🔗 与 find-skills 的关系

```
find-skills                    skill-checker
   │                               │
   │  发现候选 skill                │  评估候选 skill
   │                               │
   └──────────┬────────────────────┘
              │
              ▼
         用户决策：安装 / 跳过
```

两个 skill **独立但互补**：
- `find-skills`：负责**发现** — 搜索、推荐、安装
- `skill-checker`：负责**评估** — 检查、分析、建议

可以单独使用任一个，但搭配使用体验最佳。

---

## 🛠 技术栈

- **Claude Code Skill** — 检查流程编排
- **文件读取** — 分析 SKILL.md 及辅助文件
- **无需额外依赖** — 纯文本分析，不调用外部 API

---

## ⚠️ 注意事项

- 📖 **检查基于静态分析** — 读取 skill 文件内容进行评估，不实际运行 skill
- 🔍 **需先定位文件** — 如果 skill 尚未安装到本地，需先下载源码或通过 skills.sh 查看
- 🤖 **评估仅供参考** — 最终安装决策由用户判断
